# POC Webdriver.IO + Cucumber

Após clonar o projeto execute os comandos abaixo""

## Instalar as dependências do projeto
```
npm install
```

## Instalar selenium standalone
```
npm run selenium-postinstall
```

## Inciar o selenium
```
npm run selenium-start
```

## Executar os testes
```
npm run tests
```
